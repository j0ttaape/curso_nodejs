END-POINT
    - verbo + URL completa
    - o que é = end-point é uma funcionalidade que a API disponibiliza em um verbo junto com uma rota, cada conjunto permite que vinculamos uma funcionalidade no site (a partir da criatividade )
    - resumindo o end-point é uma funcionalidade da API
    
    
    significado dos verbos

        - GET = pegar/ler
        - POST = criar
        - PUT = substituir
        - DELETE = apagar
